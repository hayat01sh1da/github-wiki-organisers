## 1. Environment

- Ruby 3.4.5

## 2. Execution

Run the commands under `./ruby`

### 2-1. Options

|Order |Option        |Command-line Argument   |
|:-----|:-------------|:-----------------------|
|1     |group_by      |`Owner` or `Category`   |
|2     |language      |`English` or `Japanese` |
|3     |home_overflow |`true` or `false`       |
